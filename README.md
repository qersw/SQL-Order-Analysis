# SQL订单分析项目

## 项目描述
基于牛客网课程订单数据，分析2025年10月后的C++/Java/Python课程订单，识别高价值用户。

## 完成内容
- (一) 条件筛选：筛选有效订单
- (二) 分组聚合：识别购买次数≥2的优质用户
- (三) 子查询：提取优质用户完整订单信息
- (四) 聚合组合：统计用户首次购买日期和购买次数

## 文件说明
- /screenshots: 每道题的截图（题目+代码+结果）
- /code: 每道题的SQL代码文件

## 核心技能
- 多条件筛选（WHERE + IN）
- 分组聚合（GROUP BY + HAVING）
- 子查询（IN + SELECT）
- 聚合函数组合（MIN + COUNT）
